# ariaDB [![Build Status](https://travis-ci.org/proneetv/ariaDB.svg?branch=master)](https://travis-ci.org/proneetv/ariaDB)

### Build:
To build and run AriaDB service (from project root):
```shell
  $ cabal run
```

### Test:
```shell
	$ cabal run &
	$ cd src/Test
	$ runhaskell -i../Library GetPut.hs
```
